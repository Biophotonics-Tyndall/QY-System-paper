Quantum Yield Analysis

This code is working for Tm samples at 800 nm emission.

How to run

The script consists on the main file (quantumyield.m), which make the complete analysis, and subfiles for partial analysis:

  • spectra.m and spectra_ramansys.m
  • beamprofile.m
  • aucorrelation.m
  • emissionspectra.m

The subfiles and the main file don't depend on each other but they call subclasses from the loader directory:

  • SWEEP_MODE.m
  • SPECTRA.m
  • BEAM_PROFILE.m
  • CORRELATION.m
  • SPECRA_RAMANSYS.m

Input files:

  • Sweep (.csv or .txt): UCNP, its solvent (e.g.: water), Reference (Dye), its solvent

    Current (mA) Power (uW) APD (V) Pd Optical Power (uW) Set Current (mA)
    float float float float float

  • Beam profile file (.png)

Methodology

Luminescence signal → L(P)

Lucnp(P) = Lmix(P) - Lsolvent(P)
Ldye(P) = Lmix(P) - Lsolvent(P)

Absorbance → μa

mua = ln(Po / P)

Power at the centre of the cuvette → Pc

Pc = P × sqrt(eμa)

Absorbed power at the centre → Abc

Abc = μa . Pc

Experimental Quantum Yield → φexp

φexp = Lucnp(P) / Abc
