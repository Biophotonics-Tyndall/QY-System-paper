Code for the publication: Multi-variable compensated quantum yield measurements of upconverting nanoparticles with high dynamic range: a systematic approach

Access

To have access to this code and data:

  1. Download this content to your local machine,
  2. Download the data from . For storage purposes the data files were not placed on GitHub,
  3. Extract the files from data.zip to ./quantum-yield/data/ folder,
  4. Extract the files from datalake.zip to ./system-characterization/datalake/ folder

Running the scripts

Figures 3 and 4 from the published paper were generated using the system-characterization code, which was built with Matlab. Figure 5 was generate with the quantum-yield code, built with Python. Specific instructions on how to run each of them individually are available in their README files:
